## DESIGN PRINCIPLES

### 1. Glassmorphism

Use subtle glassmorphism in panels, cards, modals, sidebars, and overlays:

* translucent surfaces
* blur effects
* soft borders
* layered depth
* controlled brightness

Avoid heavy opacity that hurts readability.

### 2. Border Strategy

Use fine, elegant border lines:

* 1px neutral borders
* soft edge definition
* minimal border emphasis
* borders that support hierarchy, not dominate it

### 3. Hover Effects

Use minimal hover effects only:

* slight background shift
* tiny elevation change
* subtle border glow
* gentle icon movement if needed

No bouncing, scaling too much, or flashy effects.

### 4. Typography

Use a highly readable premium font stack.
Recommended direction:

* modern sans-serif
* strong heading presence
* clean body text
* excellent spacing between lines
* clear hierarchy

### 5. Layout

Design should rely on:

* generous whitespace
* balanced grid structure
* strong alignment
* modular sections
* clear visual grouping
* excellent information hierarchy
